## 📖 Sobre

O **Termux Customizer** é um script desenvolvido para facilitar a personalização do Termux, automatizando configurações e ajustes que normalmente exigiriam vários comandos manuais.

Seu objetivo é oferecer uma experiência mais agradável, organizada e produtiva para usuários de Android que utilizam o Termux para programação, estudos ou administração de sistemas.

---

## ✨ Funcionalidades

- 🎨 Personalização visual do terminal
- ⚡ Otimizações de configuração
- 🛠️ Automação de tarefas repetitivas
- 📦 Instalação de componentes necessários
- 🔧 Configuração simplificada
- 📱 Compatível com Android
- 🚀 Fácil instalação e utilização

---

## 📋 Requisitos

Antes de começar, certifique-se de possuir:

- Android 7.0+
- Termux atualizado
- Conexão com a internet
- Git instalado

---

## 📥 Instalação

### 1. Clonar o repositório

```bash
git clone https://github.com/recex/Recex-Site.git
```

### 2. Entrar na pasta

```bash
cd Recex-Site
```

### 3. Dar permissão de execução

```bash
chmod +x termux-customizer.sh
```

### 4. Executar

```bash
./termux-customizer.sh
```

## 🎮 Como usar

Após iniciar o script:

1. Escolha uma opção do menu.
2. Siga as instruções exibidas.
3. Aguarde a aplicação das configurações.
4. Reinicie o Termux se necessário.
